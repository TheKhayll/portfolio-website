Simple Shooting Game

Genre: Third-Person Shooting Game

Game Engine: Unreal Engine 4

Date: February 4, 2024

This is the game that i developed when i was studying the hybrid programming for Unreal Engine where it mixes Blueprint Scripting and Traditional C++ Programming, it showcases how a Simple Shooting game should be such as Movement, Animation Syncing, Combat, Bullet Tracing, Internal Sound Design.

During this development i am applying the concepts between of Hybrid Programming and it made me realize that the Hybrid programming is the style of development i want to focus in Unreal Engine since it is much easier and it allows me to scale the game and optimize it whenever it needs to.

Aiming, Line-Trace Shooting, A.I Navigations and Behaviors, Health System, Sound Designing, Environment Designing, and Directional Movement.
